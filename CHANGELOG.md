# Changelog

Все заметные изменения в проекте будут фиксироваться в этом файле.

## [Unreleased]

### Added
- команда `history export <filepath>` для экспорта истории в файл

### Changed
- интерактивный режим: при ошибке (неизвестная команда, неверное число, деление на 0) выводится сообщение и работа продолжается, а не происходит выход

## [1.0.0] - 2026-04-21

### Added
- базовый CLI-калькулятор на Python
- команды: `add`, `subtract`, `multiply`, `divide`, `power`, `modulo`
- короткие алиасы: `sum`, `sub`, `mul`, `div`, `pow`, `mod`
- интерактивный режим: `interactive` / `i`
- история операций в интерактивном режиме
- команда `clear` для очистки истории
- тесты для функций и CLI
- GitHub Actions CI
- бейдж CI в README
- контроль покрытия тестами
- обязательный порог покрытия 100%

### Quality
- форматирование через `black`
- линтинг через `ruff`
- проверка типов через `mypy`
- тесты через `pytest`
- coverage через `pytest-cov`
- pre-commit hooks

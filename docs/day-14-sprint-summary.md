## Copilot Usage Lessons

- Copilot is most useful when the next step is already clear.
- Small, specific prompts tend to produce better suggestions than broad requests.
- Suggestions still need review for correctness, style, and fit with the existing project.
- Copilot can speed up routine code and documentation work, but it should not replace understanding the code.

## Codex Usage Lessons

- Codex is useful for larger reasoning tasks such as summarizing, planning, reviewing, and connecting multiple files or decisions.
- Clear constraints matter. Stating what not to change helps keep the work focused.
- Codex should be treated as a collaborator that can inspect, explain, and propose, but its output still needs human judgment.
- The best results come from using Codex for scoped tasks with visible acceptance criteria.
